# 📊 DOE Tool 사용법 가이드 (v2.0)

## 🎯 현재 구현된 기능 현황

### ✅ **완전 구현된 분석 기능**
1. **기초 통계 분석** - 평균, 표준편차, 최솟값, 최댓값 등
2. **일원분산분석** - F-검정, 그룹 간 평균 비교
3. **이원분산분석** - 두 요인의 주효과와 상호작용 효과
4. **상관분석** - 피어슨 상관계수 매트릭스

### 🚧 **가이드만 제공되는 기능**
- 다원분산분석, 반복측정분산분석
- 단순/다중/단계적/비선형 회귀분석
- Mann-Whitney U, Kruskal-Wallis, Wilcoxon 검정
- 주성분분석, 군집분석, 판별분석

---

## 🚀 **새로운 기능: 스마트 분석 가이드**

### **🎯 두 가지 사용 방식 모두 지원**

#### **방식 1: 데이터 먼저 → 분석 선택**
1. 파일 → 데이터 가져오기
2. Excel/CSV 파일 선택
3. 분석 메뉴 선택 → 즉시 분석 실행

#### **방식 2: 분석 선택 → 데이터 준비 (권장)**
1. **분석 메뉴 먼저 선택** (데이터 없어도 OK!)
2. **스마트 가이드 창** 자동 표시
3. **"샘플 데이터 불러오기"** 버튼 클릭
4. **자동으로 적절한 샘플 데이터 로드**
5. **분석 자동 재실행**

---

## 📋 **스마트 가이드 기능**

### **🔍 현재 데이터 상태 분석**
- 데이터 행/열 수 표시
- 숫자형/그룹형 변수 개수 확인
- 분석 요구사항과 비교하여 문제점 진단

### **💡 맞춤형 해결방법 제안**
- 분석 유형에 최적화된 샘플 데이터 추천
- 데이터 형식 요구사항 상세 설명
- 단계별 해결 방법 안내

### **🚀 원클릭 해결**
- "샘플 데이터 불러오기": 적절한 샘플 자동 로드
- "직접 데이터 가져오기": 파일 선택 다이얼로그 열기
- 데이터 로드 후 분석 자동 재실행

## 🎯 **단계별 사용법 (업데이트)**

### **1단계: 애플리케이션 실행**
```bash
cd DeskTopApp
python main.py
```

### **2단계: 분석 선택 (새로운 방식)**

#### **🔢 기초 통계 분석**
```
1. 분석 → 기초 통계 선택
2. 스마트 가이드 창 표시 (데이터 없는 경우)
3. "📊 샘플 데이터 불러오기" 클릭
4. basic_statistics_sample.xlsx 자동 로드
5. 분석 자동 실행 → 결과 탭에서 확인

💡 필요한 데이터: 1개 이상의 숫자 열
📊 자동 로드 샘플: basic_statistics_sample.xlsx
```

#### **📈 상관분석**
```
1. 분석 → 회귀분석 → 상관분석 선택
2. 스마트 가이드 창 표시 (데이터 없는 경우)
3. "📊 샘플 데이터 불러오기" 클릭
4. basic_statistics_sample.xlsx 자동 로드
5. 상관계수 매트릭스 자동 계산

💡 필요한 데이터: 2개 이상의 숫자 열
📊 자동 로드 샘플: basic_statistics_sample.xlsx
```

#### **🔍 일원분산분석**
```
1. 분석 → 분산분석 → 일원분산분석 선택
2. 스마트 가이드 창 표시 (데이터 없는 경우)
3. "📊 샘플 데이터 불러오기" 클릭
4. factorial_2x3_design_categorical.xlsx 자동 로드
5. F-검정 결과 자동 계산

💡 필요한 데이터: 그룹 변수(텍스트) + 측정값(숫자)
📊 자동 로드 샘플: factorial_2x3_design_categorical.xlsx
```

#### **🔍🔍 이원분산분석**
```
1. 분석 → 분산분석 → 이원분산분석 선택
2. 스마트 가이드 창 표시 (데이터 없는 경우)
3. "📊 샘플 데이터 불러오기" 클릭
4. factorial_2x3_design_categorical.xlsx 자동 로드
5. 주효과와 상호작용 효과 자동 계산

💡 필요한 데이터: 2개의 그룹 변수(텍스트) + 1개의 측정값(숫자)
📊 자동 로드 샘플: factorial_2x3_design_categorical.xlsx
```

## 📁 **샘플 데이터 설명**

### **basic_statistics_sample.xlsx**
- **용도**: 기초 통계, 상관분석
- **구조**: Height, Weight, Age, Score, Temperature (50행)
- **특징**: Height와 Weight 간 상관관계 포함
- **자동 로드**: 기초 통계, 상관분석 선택 시

### **factorial_2x3_design_categorical.xlsx**
- **용도**: 일원분산분석, 이원분산분석
- **구조**: Group_A, Group_B, Value, ID (24행)
- **특징**: 2×3 요인설계, 그룹별 효과 포함
- **자동 로드**: 분산분석 선택 시

## ⚠️ **문제 해결 (업데이트)**

### **메뉴가 비활성화되어 있어요**
- **해결됨!** 이제 모든 분석 메뉴가 항상 활성화됩니다
- 데이터가 없어도 메뉴 선택 가능
- 스마트 가이드가 자동으로 도움을 제공합니다

### **"적절한 형식의 데이터가 필요합니다" 안내**
- **새로운 기능!** 스마트 가이드가 구체적인 문제점을 진단합니다
- 현재 데이터 상태와 요구사항을 비교 분석
- 맞춤형 해결방법을 자동으로 제안합니다

### **어떤 샘플 데이터를 써야 할지 모르겠어요**
- **자동 해결!** 분석 유형에 따라 최적의 샘플 데이터를 자동 추천
- "샘플 데이터 불러오기" 버튼으로 원클릭 로드
- 로드 후 분석이 자동으로 재실행됩니다

## 💡 **새로운 사용 팁**

### **🚀 빠른 시작 (권장)**
1. **애플리케이션 실행**
2. **원하는 분석 메뉴 바로 선택**
3. **스마트 가이드에서 "샘플 데이터 불러오기" 클릭**
4. **결과 확인**

### **📊 데이터 준비 팁**
- **열 이름**: 한글/영문 모두 가능, 공백 없이 권장
- **데이터 타입**: 숫자는 숫자로, 그룹은 텍스트로 명확히 구분
- **결측값**: 최소화하거나 공백으로 표시

### **🔍 분석 순서 권장**
1. **기초 통계** → 데이터 전반적 파악
2. **상관분석** → 변수 간 관계 탐색
3. **분산분석** → 그룹 간 차이 검증
