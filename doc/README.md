# DOE Desktop Application

실험계획법(Design of Experiments) 데스크톱 애플리케이션 - 통계 분석 및 시각화 도구

## 📋 개요

이 애플리케이션은 Minitab과 유사한 기능을 제공하는 고급 실험계획법 분석 도구입니다. Python과 PySide6를 사용하여 개발되었으며, 프로젝트 기반의 데이터 관리, 고급 통계 분석, 차트 생성 등의 기능을 제공합니다.

## ✨ 주요 기능

### 🗂️ 프로젝트 관리
- **프로젝트 생성/저장/불러오기**: JSON 형식으로 프로젝트 전체 상태 저장
- **자동 저장**: 30초마다 자동으로 프로젝트 상태 저장
- **프로젝트 탐색기**: 데이터, 분석 결과, 차트를 체계적으로 관리
- **히스토리 관리**: 모든 분석 및 차트 생성 기록 보존

### 📊 데이터 관리
- **다양한 파일 형식 지원**: CSV, Excel (xlsx, xls) 파일 가져오기/내보내기
- **실시간 데이터 편집**: 스프레드시트 형태의 직관적인 데이터 편집
- **데이터 요약**: 자동 통계 요약 및 열별 상세 정보 제공
- **샘플 데이터**: 실험설계 관련 샘플 데이터 파일 포함

### 📊 고급 데이터 관리
- **데이터 입출력**: CSV, Excel (xlsx, xls) 파일 가져오기/내보내기, 클립보드 연동
- **데이터 편집**: 스프레드시트 형태의 직관적인 데이터 편집, 행/열 추가/삭제, 정렬
- **데이터 변환**: 변수명 변경, 데이터 타입 변경, 파생변수/더미변수 생성, 로그 변환, 표준화, 정규화
- **데이터 품질**: 결측값 처리, 이상값 탐지, 중복값 제거, 데이터 검증
- **데이터 뷰**: 데이터 요약, 미리보기, 필터링, 변수 정보 상세 보기
- **샘플 데이터**: 실험설계 관련 샘플 데이터 파일 포함

### 📈 통계 분석
- **기초 통계**: 평균, 표준편차, 최소/최대값, 사분위수 등
- **상관분석**: 피어슨 상관계수 및 상관행렬 분석
- **ANOVA 분석**: 일원/다원 분산분석 (구현 진행 중)
- **회귀분석**: 선형회귀 및 다중회귀분석 (구현 진행 중)
- **실험설계**: 요인설계 생성 (계획 중)

### 📈 고급 통계 분석
- **기초 통계**: 평균, 표준편차, 최소/최대값, 사분위수 등
- **상관분석**: 피어슨 상관계수 및 상관행렬 분석
- **분산분석(ANOVA)**: 일원분산분석, 이원분산분석, 다원분산분석, 반복측정분산분석
- **회귀분석**: 단순회귀분석, 다중회귀분석, 단계적회귀분석, 비선형회귀분석
- **비모수검정**: Mann-Whitney U 검정, Kruskal-Wallis 검정, Wilcoxon 부호순위 검정
- **다변량분석**: 주성분분석(PCA), 군집분석, 판별분석
- **종합분석실행**: 전체 분석 파이프라인 자동 실행

### 📊 시각화
- **히스토그램**: 단일 변수의 분포 시각화
- **박스플롯**: 변수의 분포와 이상치 탐지
- **산점도**: 두 변수 간의 관계 분석
- **선 그래프**: 시계열 데이터 시각화
- **막대 그래프**: 범주별 데이터 비교
- **상관행렬 히트맵**: 변수 간 상관관계 시각화
- **인터랙티브 차트**: Plotly를 활용한 동적 차트 지원

### 🧪 실험설계(DOE)
- **스크리닝 설계**: 완전요인설계, 부분요인설계, Plackett-Burman 설계
- **최적화 설계**: 반응표면설계(RSM), Box-Behnken 설계, 중심합성설계(CCD)
- **견고성 설계**: 직교배열설계, 다구치설계
- **특수 설계**: 혼합설계, 분할구설계
- **사용자 정의 설계**: 커스텀 실험점 정의

### 📊 고급 시각화
- **DOE 차트**: 주효과도, 상호작용도, 등고선도, 3D 표면도, 파레토 차트
- **진단 차트**: 정규확률도(Q-Q Plot), 잔차도
- **기본 차트**: 산점도, 히스토그램, 박스플롯, 선 그래프, 막대 그래프
- **상관행렬 히트맵**: 변수 간 상관관계 시각화
- **인터랙티브 차트**: Plotly를 활용한 동적 차트 지원

### 🎨 사용자 인터페이스
- **모던한 UI**: Qt6 기반의 세련된 사용자 인터페이스
- **반응형 레이아웃**: 화면 크기에 따른 자동 조정
- **한글 폰트 지원**: 완전한 한국어 지원
- **고해상도 디스플레이 지원**: HiDPI 스케일링 지원
- **메모리 사용량 모니터링**: 실시간 메모리 사용량 표시

## 🚀 설치 및 실행

### 1. 시스템 요구사항

- **Python**: 3.8 이상
- **운영체제**: Windows 10/11, macOS, Linux
- **메모리**: 최소 4GB RAM 권장
- **저장공간**: 500MB 이상

### 2. 의존성 설치

```bash
# 가상환경 생성 (권장)
python -m venv venv

# 가상환경 활성화
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate

# 의존성 설치
pip install -r requirements.txt
```

### 3. 애플리케이션 실행

```bash
python main.py
```

## 📖 사용법

### 프로젝트 시작하기

1. **새 프로젝트 생성**: `파일` → `새 프로젝트`
2. **기존 프로젝트 열기**: `파일` → `프로젝트 열기`
3. **프로젝트 저장**: `Ctrl+S` 또는 `파일` → `프로젝트 저장`

### 데이터 가져오기 및 편집

1. **데이터 가져오기**:
   - `파일` → `데이터 가져오기` 선택
   - CSV 또는 Excel 파일 선택
   - 데이터가 자동으로 테이블에 표시

2. **데이터 편집**:
   - 데이터 탭에서 직접 셀 클릭하여 편집
   - 행/열 추가/삭제 버튼 사용
   - 우클릭 메뉴로 고급 옵션 사용

### 통계 분석 수행

1. **프로젝트 탐색기**에서 `분석` 탭 선택
2. **빠른 분석** 버튼 사용:
   - 📊 기초 통계
   - 🔗 상관분석
   - 🧪 ANOVA (구현 진행 중)
   - 📈 회귀분석 (구현 진행 중)
3. 분석 결과는 자동으로 히스토리에 저장

### 고급 통계 분석 수행

1. **분석 메뉴** 활용:
   - **기초 통계**: 변수의 기본 통계량 계산
   - **분산분석**: 일원/이원/다원/반복측정 분산분석
   - **회귀분석**: 단순/다중/단계적/비선형 회귀분석
   - **비모수검정**: Mann-Whitney, Kruskal-Wallis, Wilcoxon 검정
   - **다변량분석**: PCA, 군집분석, 판별분석
2. **종합분석실행**으로 전체 분석 파이프라인 자동 실행
3. 분석 결과는 자동으로 히스토리에 저장되며 결과 탭에서 확인

### 실험설계(DOE) 수행

1. **실험설계 메뉴** 활용:
   - **스크리닝 설계**: 완전요인, 부분요인, Plackett-Burman 설계
   - **최적화 설계**: RSM, Box-Behnken, 중심합성설계(CCD)
   - **견고성 설계**: 직교배열, 다구치설계
   - **특수 설계**: 혼합설계, 분할구설계
2. 설계 마법사를 통해 실험 조건 설정
3. 생성된 실험설계는 데이터 탭에 자동 표시

### 차트 생성

1. **차트 탭** 선택
2. 좌측 패널에서 차트 유형 선택
3. X축, Y축 변수 설정
4. **차트 생성** 버튼 클릭
5. 생성된 차트는 차트 히스토리에 자동 저장

### 고급 차트 생성

1. **차트 메뉴** 활용:
   - **기본 차트**: 산점도, 히스토그램, 박스플롯, 선 그래프, 막대 그래프
   - **DOE 차트**: 주효과도, 상호작용도, 잔차도, 등고선도, 3D 표면도, 파레토 차트
   - **진단 차트**: 정규확률도(Q-Q Plot), 상관행렬 히트맵
2. 차트 유형별 맞춤 설정 옵션 제공
3. 생성된 차트는 인터랙티브하게 확대/축소/이동 가능
4. **모든 차트 지우기**로 차트 히스토리 초기화

### 데이터 품질 관리

1. **데이터 메뉴** 활용:
   - **데이터 입출력**: 파일 가져오기/내보내기, 클립보드 복사/붙여넣기
   - **데이터 편집**: 행/열 추가/삭제, 데이터 정렬, 찾기/바꾸기
   - **데이터 변환**: 변수명 변경, 데이터 타입 변경, 파생변수 생성
   - **데이터 품질**: 결측값/이상값/중복값 처리, 데이터 검증
   - **데이터 뷰**: 데이터 요약, 미리보기, 필터링
2. **도구 메뉴**에서 고급 데이터 변환 기능 활용
3. 처리 결과는 실시간으로 데이터 탭에 반영

- **데이터 편집**: 행/열 편집, 정렬, 찾기/바꾸기
- **데이터 변환**: 변수 관리, 파생변수 생성, 로그 변환, 표준화, 정규화
- **데이터 품질**: 결측값/이상값/중복값 처리, 데이터 검증
- **데이터 뷰**: 요약, 미리보기, 필터링, 변수 정보

- **견고성 설계**: 직교배열, 다구치설계
- **특수 설계**: 혼합설계, 분할구설계
- **사용자 정의 설계**

- **회귀분석**: 단순/다중/단계적/비선형
- **비모수검정**: Mann-Whitney, Kruskal-Wallis, Wilcoxon
- **다변량분석**: PCA, 군집분석, 판별분석

## 🔧 문제 해결

### 한글 폰트 문제
차트에서 한글이 깨져 보이는 경우:
```python
# 폰트 캐시 재생성
import matplotlib.font_manager
matplotlib.font_manager._rebuild()
```

### 메모리 사용량 표시 안됨
`psutil` 라이브러리 설치 확인:
```bash
pip install psutil
```

### 고해상도 디스플레이 문제
환경 변수 설정:
```bash
export QT_AUTO_SCREEN_SCALE_FACTOR=1
export QT_ENABLE_HIGHDPI_SCALING=1
```
