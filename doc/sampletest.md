sample2.xlsx -> DOE 분석 > 스크리닝 분석 > 부분요인 ANOVA
               -> 차트: 주효과도(요인=F1..F4, Y=Response), 상호작용도(예: X=F1, 그룹=F2, Y=Response), 박스/막대/산점도
rsm_sample.xlsx -> DOE 분석 > 최적화 분석 > 반응표면분석 (RSM)
                  -> 차트: 주효과도/상호작용도, 산점도(X1 vs Response, 그룹=X2), 상관행렬
boxbehnken_sample.xlsx -> DOE 분석 > 최적화 분석 > Box-Behnken 분석
                         -> 차트: 주효과도/상호작용도, 산점도(A vs Response, 그룹=B/C), 상관행렬
ccd_sample.xlsx -> DOE 분석 > 최적화 분석 > CCD 분석
                  -> 차트: 주효과도/상호작용도, 산점도(X1 vs Response, 그룹=X2), 상관행렬
orthogonal_sample.xlsx -> DOE 분석 > 견고성/직교배열 분석 > 직교배열 분석
                         -> 차트: 주효과도/상호작용도(요인=Factor1..3, Y=Response), 박스/막대
orthogonal_sample.xlsx -> DOE 분석 > 견고성/직교배열 분석 > 다구치(S/N) 분석 (동일 데이터 예시)
                         -> 차트: 주효과도/상호작용도, 박스/막대
mixture_sample.xlsx -> DOE 분석 > 특수/기타 분석 > 혼합 설계 분석
                       -> 차트: 주효과도(요인=CompA..C, Y=Response), 산점도(두 성분 vs Response)
splitplot_sample.xlsx -> DOE 분석 > 특수/기타 분석 > 분할구 설계 분석
                         -> 차트: 주효과도/상호작용도(Whole_A, Sub_B), 박스/막대
