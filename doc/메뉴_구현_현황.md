## 🎯 **우선순위별 개발 계획**

### **우선순위 1 (즉시 필요)**
1. **단순회귀분석** - 상관분석 기반으로 확장
2. **다중회귀분석** - 단순회귀 확장
3. **산점도 차트** - 시각화 기본 기능
4. **히스토그램** - 분포 확인용

### **우선순위 2 (다음 버전)**
1. **데이터 편집 기능** (행/열 삽입/삭제, 정렬)
2. **기본 차트** (상자그림, 막대그래프)
3. **데이터 변환** (로그변환, 표준화)
4. **비모수 검정** (Mann-Whitney, Kruskal-Wallis)

### **우선순위 3 (장기 계획)**
1. **DOE 설계 생성** (완전요인, 부분요인)
2. **고급 차트** (주효과, 상호작용 그림)
3. **다변량 분석** (PCA, 군집분석)
4. **데이터 품질 관리** (결측값, 이상값 처리)
