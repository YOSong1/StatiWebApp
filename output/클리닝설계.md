# 클리닝설계 메뉴 설계 방향

## 1. 기능 개요
- 실험설계 메뉴 하위에 "클리닝설계" 메뉴를 추가한다.
- 사용자가 업로드한 엑셀 데이터를 기반으로 실험 클리닝 계획을 수립하고 분석 결과를 제공한다.

## 2. 데이터 입력 및 스키마 처리
- 사용자가 엑셀 파일을 업로드하면 자동으로 컬럼명, 데이터 유형, 누락값 여부 등을 파악하여 스키마를 생성한다.
- 생성된 스키마와 샘플 데이터를 테이블 형태로 화면에 표시한다.
- 사용자는 필요에 따라 실험에 사용할 변수 및 수준을 선택하거나 변환할 수 있다.

## 3. 실험 설계 흐름
1. **데이터 확인 단계**: 스키마와 데이터를 확인하고 전처리 옵션을 선택.
2. **설계 생성 단계**: 요인(variables)과 수준(levels)을 지정하여 실험 설계를 생성한다. 기본적으로 요인별 2수준 요인배치법이나 필요한 경우 반응표면법 등을 적용할 수 있도록 한다.
3. **실험 배치 확인**: 생성된 설계표를 미리보기 형태로 제공하고, 필요 시 실험 순서를 랜덤화할 수 있는 옵션을 포함한다.
4. **분석 및 클리닝 조건 정의**: 실험 데이터를 기반으로 클리닝 조건(예: 온도, 시간, 용량 등)을 분석해 최적 조건을 도출한다.

## 4. 분석 결과 및 시각화
- 실험 결과 데이터를 이용해 ANOVA, 회귀분석 등 통계적 분석을 수행한다.
- 주요 요인의 효과, 상호작용 효과 등을 시각화(주효과도, 상호작용도, 반응표면 그래프 등)하여 제공한다.
- 분석 결과에 대한 요약 및 해석을 텍스트 형태로 제공하여 사용자 이해를 돕는다.

## 5. 결과 출력 및 활용
- 분석 결과와 그래프를 PDF 또는 이미지로 내보낼 수 있는 기능을 제공한다.
- 설계 및 분석 과정에서 사용된 매개변수를 저장하여 재사용할 수 있도록 한다.

## 6. 추후 확장 고려사항
- 다양한 데이터 소스(CSV, DB 등) 지원 확대.
- 고급 실험설계 기법(혼합 설계, 부분 요인 설계 등) 추가.
- 자동 보고서 생성 기능.
